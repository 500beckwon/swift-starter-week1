## 2.주기억 장치는 왜 주기억 장치라고 불릴까요?

보조기억장치(HDD)는 수많은 기억들을 보관하는 창고라고 생각하고 
주기억장치(RAM)는 현재 cpu에서 수행하려 하는 주된 것들을 기억하고 있어서 
주기억 장치라고 생각합니다.
